Quick start

Flash & set up your observer

Get a MeshCore Observer node on the air in a few minutes. Everything is done in the browser — no drivers or command-line tools required.

info

Flashing needs the Web Serial API — use Chrome or Edge on a desktop computer. Phones and Safari can't flash.

Flash the firmware

  1. Connect your board

    Plug your MeshCore device into your computer with a USB data cable.

  2. Pick your device & role

    In the flasher, choose your board, then select the Observer role. Use the filter box if you have a long list of devices.

  3. Choose a version and flash

    Leave the version on stable (or opt into a dev build), then click Flash! and grant serial access when prompted. Keep the device connected until it reports Flashing complete.

    Updating an existing device? Leave Erase device unchecked so your identity and settings are preserved.

Configure your observer

After the first flash, set the radio parameters, a device name, WiFi credentials, and your MQTT broker(s). The quickest way is over USB: click Console in the header to open the serial console, then run these commands (substitute your own values): 

set radio 910.525,62.5,7,5
set tx 22
set name MyObserver
set mqtt.iata XXX
set wifi.ssid YourWiFiNetwork
set wifi.pwd YourWiFiPassword
reboot

Radio settings must match your regional mesh — the values above are only an example. Replace XXX with the 3-letter IATA code of your nearest airport. See the full guide for timezone, additional MQTT presets, and commands to verify the connection.

chevron_right First-boot web configuration Beta

A newer, cable-free option: on first boot (or after a full erase) the observer starts its own WiFi hotspot and serves a configuration page. This is still in beta — if anything misbehaves, fall back to the USB console commands above.

  1. Join the setup hotspot

    Power on the device and, within its first few minutes, open the WiFi list on your phone or computer. Join the network named MeshCore-Setup-XXXX (the suffix is unique to your device).

  2. Open the configuration page

    Your device should open the setup page automatically. If it doesn't, browse to http://192.168.4.1 in any browser.

  3. Enter your settings and save

    Fill in your radio parameters, device name, WiFi network, and MQTT broker preset(s), then save. The observer reboots, joins your WiFi, and starts uplinking mesh traffic.
